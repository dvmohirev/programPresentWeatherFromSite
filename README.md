# programPresentWeatherFromSite
Мастер-класс. Парсим погоду с сайта и выводим у себя в консоль.<br>
Темы, знание которых мы укрепим с помощью программирования приложения:
- библиотека Jsoup;
- CSS Query Language;
- обработка исключений;
- ООП;
- циклы;
- массивы;
- списки;
- условные операторы;
- работа со строками;
- вывод на консоль.
